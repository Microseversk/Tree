# Создание узла

-   Пишем название в input`e
-  Выделяем родительский узел
-  Нажимаем "Создать узел"

# Удаление узла

-  Выделяем узел
-  Нажимаем "Удалить узел"

# Переименование узла

-  Пишем название в input`e
-  Выделяем узел
-  Нажимаем "Переименовать узел"

# Сбросить до первоначального вида

-  Нажимаем "Сбросить"
